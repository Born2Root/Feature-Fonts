Friendly Font - Demo

What is this font?

The "friendly Font" is an self censoring intelligent typeface. It automatically removes unfriendly words and replaces them with random characters. The user sees the censored version, but underlying the real uncensored text is still available.

How to use?

In the README of the repository you can find examples on how to use the font on several applications supporting OpenType font features.
Also you can read on how the font was made and how you can create your own font in the same style.
